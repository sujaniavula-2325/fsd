A> Water Consumption

Recently, Chef visited his doctor. The doctor advised Chef to drink at least 2000 ml of water each day.

Chef drank X ml of water today. Determine if Chef followed the doctor’s advice or not.

Input Format
The first line contains a single integer T — the number of test cases.
The first and only line of each test case contains one integer X.

Output Format
For each test case, output YES if Chef followed the doctor’s advice. Otherwise, output NO.

Inputcopy Outputcopy
3
2999
1450
2000
YES
NO
YES

SOLUTION:
import java.util.Scanner;
public class Main{
public static void main(String[] args){
Scanner sc=new Scanner(System.in);
int T=sc.nextInt();
for(int i=0;i<T;i++){
int X=sc.nextInt();
if(X>=2000)
System.out.println("YES");
else
System.out.println("NO");
}
}
}

B> Non-Negative Product

Alice has an array of N integers. She wants the product of all elements to be non-negative.

Input Format
First line contains T.
For each test case:
First line contains N.
Second line contains N integers.

Output Format
Print minimum number of elements to remove.

Inputcopy Outputcopy

4

4
2 -1 0 100
4
2 -1 -1 100
0
1
0
0

SOLUTION:
import java.util.Scanner;
public class Main{
public static void main(String[] args){
Scanner sc=new Scanner(System.in);
int T=sc.nextInt();
while(T-->0){
int N=sc.nextInt();
int neg=0;
boolean zero=false;
for(int i=0;i<N;i++){
int x=sc.nextInt();
if(x<0) neg++;
if(x==0) zero=true;
}
if(zero || neg%2==0)
System.out.println(0);
else
System.out.println(1);
}
}
}

U> Card Game

Input
First line contains n.
Next n lines contain two strings.

Output
Print final scores.

Inputcopy Outputcopy
3
cat dog

SOLUTION:
import java.util.Scanner;
public class Main{
public static void main(String[] args){
Scanner sc=new Scanner(System.in);
int n=sc.nextInt();
int taro=0,hanako=0;
for(int i=0;i<n;i++){
String a=sc.next();
String b=sc.next();
int cmp=a.compareTo(b);
if(cmp>0) taro+=3;
else if(cmp<0) hanako+=3;
else{ taro++; hanako++; }
}
System.out.println(taro+" "+hanako);
}
}

W> Standard Deviation

Input
Multiple datasets until n=0.

Output
Print standard deviation.

Inputcopy Outputcopy
5
70 80 100 90 20
3
80 80 80
0
27.85677655
0.00000000

SOLUTION:
import java.util.Scanner;
public class Main{
public static void main(String[] args){
Scanner sc=new Scanner(System.in);
while(true){
int n=sc.nextInt();
if(n==0) break;
double[] a=new double[n];
double sum=0;
for(int i=0;i<n;i++){
a[i]=sc.nextDouble();
sum+=a[i];
}
double mean=sum/n;
double var=0;
for(int i=0;i<n;i++){
double d=a[i]-mean;
var+=d*d;
}
var/=n;
System.out.printf("%.8f%n",Math.sqrt(var));
}
}
}

for(int f=0;f<3;f++){
for(int r=0;r<10;r++){
System.out.print(" "+a[b][f][r]);

}
