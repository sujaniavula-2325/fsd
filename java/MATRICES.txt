MATRICES

1> Write a program to read a matrix and print it.

Input Format
First line contains two integers R and C (number of rows and columns).
Next R lines contain C integers each.

Output Format
Print the matrix elements row-wise.

2 3
1 2 3
4 5 6

4 5 6
