1> Write a program which prints 1000 "Hello World".

Input
There is no input for this problem.

Output
The output consists of 1000 lines. Print "Hello World" in each line.

Inputcopy Outputcopy
No input
Hello World
Hello World
Hello World
...
Hello World

SOLUTION:
public class Main{
public static void main(String[] args){
for(int i=0;i<1000;i++){
System.out.println("Hello World");
}
}
}

2> Write a program which prints small/large/equal relation of given two integers a and b.

Input
Two integers a and b separated by a single space.

Output
a < b if a is less than b
a > b if a is greater than b
a == b if a equals b

Inputcopy Outputcopy
1 2
a < b

Inputcopy Outputcopy
4 3
a > b

Inputcopy Outputcopy
5 5
a == b

SOLUTION:
import java.util.Scanner;
public class Main{
public static void main(String[] args){
Scanner sc=new Scanner(System.in);
int a=sc.nextInt();
int b=sc.nextInt();
if(a<b)
System.out.println("a < b");
else if(a>b)
System.out.println("a > b");
else
System.out.println("a == b");
}
}

8> Compare prices of two TVs after discount and find cheaper one.

Input Format
First line contains T.

Output Format
Print First, Second or Any.

Inputcopy Outputcopy
3
85 75 35 20
100 99 0 0
30 40 0 10
First
Second
Any

SOLUTION:
import java.util.Scanner;
class Main{
public static void main(String[] args){
Scanner sc=new Scanner(System.in);
int T=sc.nextInt();
for(int i=0;i<T;i++){
int A=sc.nextInt();
int B=sc.nextInt();
int C=sc.nextInt();
int D=sc.nextInt();
int p1=A-C;
int p2=B-D;
if(p1<p2)
System.out.println("First");
else if(p2<p1)
System.out.println("Second");
else
System.out.println("Any");
}
}
}

Print minimum packets.

Inputcopy Outputcopy
4
20 12

public static void main(String[] args){

int N=sc.nextInt();
